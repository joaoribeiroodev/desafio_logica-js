# 🚗 Calculadora de Consumo de Combustível

**Desafio de Lógica em JavaScript**

Uma aplicação interativa que calcula o gasto diário com combustível baseado em distância percorrida, consumo do veículo e análise de preços em postos de gasolina.

---

## 📋 Sobre o Projeto

Este projeto é um **desafio educacional de programação** que demonstra conceitos fundamentais de JavaScript através de uma solução prática e útil. O programa coleta informações do usuário, realiza cálculos matemáticos e apresenta resultados de forma clara e organizada.

### 🎯 Objetivo

Aplicar conceitos essenciais de lógica de programação em JavaScript:
- ✅ Interação com o usuário via `prompt()`
- ✅ Validação de entrada de dados
- ✅ Estruturas de repetição (`for` loop)
- ✅ Operações matemáticas
- ✅ Manipulação de DOM e console
- ✅ Formatação de valores numéricos

---

## ✨ Funcionalidades

### 📥 Coleta de Dados
- Distância entre casa e trabalho (em km)
- Consumo médio do veículo (km/L)
- Quantidade de postos pesquisados
- Preços de combustível em cada posto

### 🧮 Cálculos Realizados
| Cálculo | Descrição |
|---------|-----------|
| **Consumo Necessário** | Litros necessários para percorrer a distância |
| **Menor Preço** | Identificação do melhor preço entre postos |
| **Média de Preços** | Valor médio encontrado na pesquisa |
| **Gasto Diário** | Custo total ida e volta com melhor preço |

### 📊 Exibição de Resultados
- **Página HTML**: Resultados formatados com 2 casas decimais
- **Console**: Dados brutos para debug e análise
